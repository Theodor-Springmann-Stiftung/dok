1001SpielkarteEine Umfangseinheit einer Karte, die mit Wörtern, Zahlen oder Bildern bedruckt ist, und von einer Person oder einer Gruppe als Grundlage für die Durchführung einer spezifischen Aktivität verwendet wird.
1002AtlasEine Umfangseinheit eines Bandes von Karten oder sonstigen kartografischen Inhalten mit oder ohne beschreibenden Text.
1003BehältnisEine Umfangseinheit, die aus einer Schachtel besteht, die gebundene oder ungebundene Ressourcen enthält.
1004SchaubildEine Umfangseinheit, die aus einer zweidimensionale Darstellung der Daten in grafischer oder tabellarischer Form besteht.
1005MünzeEine Umfangseinheit, die aus einem Stück Metall besteht, das von einer staatlichen Behörde zur Verwendung als Geld geprägt wurde.
1006CollageEine Umfangseinheit, die aus einem zweidimensionalen oder sehr flachen reliefartigen Werk besteht, das durch Befestigung von Papier, Stoff, Fotografien oder sonstigen Materialien auf einer ebenen Oberfläche gemacht ist.
1007SpalteEine Umfangseinheit, die aus einem von zwei oder mehreren vertikalen Textabschnitten besteht, die auf derselben Seite oder demselben Blatt erscheinen.
1008DiagrammEine Umfangseinheit, die aus einer geografischen Darstellung numerischer Daten beziehungsweise dem Verlauf oder den Ergebnissen einer Tätigkeit oder eines Prozesses besteht.
1009DioramaEine Umfangseinheit, die aus einer dreidimensionalen Darstellung einer Szene besteht, die durch das Platzieren von Objekten, Figuren und so weiter vor einem zweidimensional gezeichneten Hintergrund geschaffen wird.
1010ZeichnungEine Umfangseinheit, die aus einem visuellen Werk besteht, das durch Zeichnen, das heißt Aufbringen von Linien auf eine Oberfläche, häufig Papier, unter Verwendung eines Bleistifts, eines Zeichenstifts, Kohle oder eines anderen Spuren hinterlassenden Instruments erstellt wird. Dabei liegt der Fokus auf der Darstellung der Form und weniger auf der Anwendung von Farbe.
1011AusstellungsstückEine Umfangseinheit, die aus ausgestellten Gegenständen mit dem Ausstellungsumfeld (Kisten, Labels und so weiter) besteht.
1012LernkarteEine Umfangseinheit, die aus einer Karte (oder der digitalen Repräsentation einer Karte) besteht, die Wörter, Zahlen, Bilder und so weiter enthält und für eine schnelle Anzeige als Lernhilfe konzipiert ist.
1013Gefalteter BogenEine Umfangseinheit, die aus einem Bogen besteht, der so angeordnet ist, dass er nach dem Falten seitenweise gelesen werden kann.
1014SpielEine Umfangseinheit, die aus einem Satz von Gegenständen besteht, der für die Bedienung nach vorgeschriebenen oder impliziten Regeln für Unterricht, Unterhaltung oder Therapie konzipiert ist.
1015GlobusEine Umfangseinheit, die aus einer Darstellung der Erde oder eines anderen Himmelskörpers (real oder imaginär) auf der Oberfläche einer Kugel besteht.
1016IkoneEine Umfangseinheit, die aus einem Bild besteht, das eine heilige Entität porträtiert, und das selbst als heilig angesehen wird.
1017PuzzleEine Umfangseinheit, die aus einem Bild besteht, normalerweise auf Karton oder Holz, das in ineinandergreifende Formen geschnitten wird, die wieder zusammengefügt werden sollen.
1018BlattEine Umfangseinheit, die aus einem einzelnen gebundenen oder befestigten Bogen als Untereinheit eines Bandes bestehend. Jedes Blatt besteht aus zwei Seiten, auf jeder Seite eine, die auch leer sein können.
1019KarteEine Umfangseinheit, die aus einer Darstellung (normalerweise maßstabsgerecht und auf einem zweidimensionalen Medium) einer Auswahl von Material oder abstrakten Besonderheiten auf oder in Verbindung mit der Erdoberfläche, einem anderen Himmelskörper oder einem imaginären Ort besteht.
1020MedailleEine Umfangseinheit, die aus einem kleinen Stück Metall besteht, das ein-oder beidseitig mit einem Relief versehen ist und einen Erinnerungszweck hat. Sie wird nicht als Zahlungsmittel verwendet.
1021NachbildungEine Umfangseinheit, die aus einer physischen Darstellung eines Gerätes oder eines Prozesses besteht, das/der zur Übung oder zur Analyse modifiziert werden kann, um einen bestimmten Teil oder eine Funktion hervorzuheben.
1022ModellEine Umfangseinheit, die aus einer dreidimensionalen Darstellung eines realen oder imaginären Objekts besteht.
1023SeiteEine Umfangseinheit, die aus einer einzelnen Seite eines Blatts besteht.
1024GemäldeEine Umfangseinheit, die aus einem Objekt besteht, bei dem Bilder in erster Linie durch das direkte Auftragen von Pigment, das in ein Medium eingebracht ist und in Farbmengen auf einer im Allgemeinen zweidimensionalen Oberfläche angeordnet wird, dargestellt werden.
1025FotografieEine Umfangseinheit, die aus einem Bild besteht, das mit einer Linse aufgenommen und auf die Oberfläche eines Blatts übertragen wird.
1026BildEine Umfangseinheit, die aus zweidimensionalen Darstellungen besteht.
1027MappeEine Umfangseinheit, die aus einem Behältnis zum Zusammenhalten von losen Materialien besteht, das normalerweise aus zwei Einbanddecken besteht, die an der Rückseite verbunden sind.
1028PostkarteEine Umfangsangabe, die aus einer Karte besteht, auf die eine Nachricht geschrieben oder gedruckt werden kann, um sie ohne Umschlag zu versenden.
1029PlakatEine Umfangseinheit, die aus einer normalerweise dekorativen oder bildlichen Ankündigung besteht, die dazu gedacht ist, eine Aktivität, einen Anlass, ein Produkt oder eine Dienstleistung zu bewerben, zu annoncieren, zu fördern oder zu propagieren. Dazu zählt auch ein dekorativer, in Massenproduktion hergestellter Druck, der zum Aufhängen gedacht ist.
1030DruckgrafikEine Umfangseinheit, die aus einem bildhaften Werk besteht, das durch die Übertragung eines Bilds mittels einer Matrix, zum Beispiel einer Platte, einem Block oder einem Bildschirm, unter Verwendung von beliebigen Druckprozessen erstellt wird.
1031ProfilEine Umfangseinheit einer maßstabsgetreuen Repräsentation der Schnittpunkte einer vertikalen Oberfläche (die eine Ebene sein kann oder nicht) mit der Erdoberfläche oder der Schnittpunkte einer solchen vertikalen Oberfläche mit der eines konzeptionellen dreidimensionalen Modells, welches Phänomene repräsentiert, die eine kontinuierliche Verbreitung haben.
1032RöntgenbildEine Umfangseinheit, die aus einer Fotografie besteht, die durch den Durchtritt von Strahlung, wie zum Beispiel Röntgenstrahlen, Gammastrahlen oder Neutronen, durch ein lichtundurchlässiges Objekt erzeugt wird.
1033FernerkundungsbildEine Umfangseinheit, die aus einem bildlichen Produkt eines Fernerkundungsinstruments besteht, das reflektierte und/oder emittierte elektromagnetische Strahlung aus der Distanz entdeckt und misst sowie im Falle eines Sonars Schallwellen unter Wasser reflektiert.
1034SkulpturEine Umfangseinheit, die aus einer physischen Darstellung besteht, normalerweise in der Kunst, bei der ein Bild oder eine Form als Relief, als Intaglio oder in Rundumansicht erzeugt werden.
1035SchnittEine Umfangseinheit einer maßstabsgetreuen Darstellung einer vertikalen Oberfläche (normalerweise eine Ebene), die sowohl das Profil anzeigt, wo es die Oberfläche eines Himmelskörpers oder eines konzeptionellen Modells schneidet, als auch die darunter liegenden Strukturen entlang der Schnittebene.
1036MusterEine Umfangseinheit, die aus einer einzelnen Einheit oder einem Beispiel besteht, das ausgewählt wurde, um einen größeren Bestand oder eine Aggregation zu repräsentieren.
1037StudiendruckEine Umfangseinheit, die aus einem Bild besteht, das manchmal von einem Text begleitet wird und für die Ausbildung verwendet wird.
1038Technische ZeichnungEine Umfangseinheit, die aus einem Querschnitt, einem Detail, einem Diagramm, einem Aufriss, einer Ansicht, einem Plan, einem Arbeitsplan und so weiter besteht, der/die für die Verwendung in einem technischen oder anderen fachlichen Zusammenhang gemacht wird.
1039SpielzeugEine Umfangseinheit, die aus einem Gegenstand besteht, der für die Bildung, Unterhaltung oder Stimulation durch Spiel entwickelt wird.
1040AnsichtEine Umfangseinheit einer perspektivische Darstellung der Landschaft, in der Einzelheiten wie auf eine schräge Ebene projiziert dargestellt sind.
1041SchautafelEine Umfangseinheit, die aus einer tabellarischen oder grafischen Darstellung von Daten besteht, die sich für die Anzeige an einer Wand eignet.
