T1001EinbandEine Zusammenstellung von einem Blatt oder mehreren Blättern, mit oder ohne aufgeklebten äußeren Umschlag, in einer festen Reihenfolge.
T1004CartridgeEin rechteckiges Gehäuse, das mit einer einzigen Spule ausgestattet ist, die eine Rolle hält.
T1005KassetteEin rechteckiges Gehäuse, das mit zwei Spulen ausgestattet ist, die eine Rolle halten.
T1002FlipchartEine Hängevorrichtung besteht, die mehrere Blätter hält, die zur Verwendung an einer Staffelei konzipiert sind.
T1006nicht zutreffendDas Speichermedium ist nicht von einem Gehäuse umschlossen.
T1003SpuleEine geflanschte Spule zur Aufnahme einer Rolle.
