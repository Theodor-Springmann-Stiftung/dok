1002Integrierender, zeitlich begrenzter ErweiterungsplanEin Erweiterungsplan für ein Werk, dessen Realisierung in einer unterscheidbaren Expression vorgesehen ist, die in einem begrenzten Zeitraum verkörpert wird.
1003Integrierender, zeitlich unbegrenzter ErweiterungsplanEin Erweiterungsplan für ein Werk, dessen Realisierung in einer unterscheidbaren Expression vorgesehen ist, die in einem unbegrenzten Zeitraum verkörpert wird.
1001Statischer ErweiterungsplanEin Erweiterungsplan für ein Werk, dessen Realisierung in einer unterscheidbaren Expression oder mehreren unterscheidbaren Expressionen vorgesehen ist, die alle zeitgleich verkörpert werden.
1004Sukzessiver, zeitlich begrenzter ErweiterungsplanEin Erweiterungsplan für ein Werk, dessen Realisierung in mehreren unterscheidbaren Expressionen vorgesehen ist, die in einem begrenzten Zeitraum verkörpert werden.
1005Sukzessiver, zeitlich unbegrenzter ErweiterungsplanEin Erweiterungsplan für ein Werk, dessen Realisierung in mehreren unterscheidbaren Expressionen vorgesehen ist, die in einem unbegrenzten Zeitraum verkörpert werden.
